Lesson: Recursion

Recursion
A recursive method is a method which calls itself. When the method calls itself, it is called a recursive method. Recursive methods have one or more base cases (termination cases), and a recursive case. If the base case is met, there is no recursive call but if the recursive case is met, the method is called again doing the same task altered to "be smaller" or processing less. Without a termination case, the recursive method would be called infinitely and never return the desired result. For example, a common recursive method is a reverse string method, which for each recursive call the string will be one character smaller and then when the string is 1 character long (the base case), that character will be returned.

Iteration
Recursive solutions can often have the same result as iterative solutions. However, not all problems can be solved iteratively. Problems that are not linear will often only be able to be solved with a recursive solution.
