Lesson: Arrays

Arrays
An array is a consecutive space of memory that holds multiple values of the same data type. Arrays are treated like objects and can hold primitive data or objects.The individual locations in the array are called the elements of the array. An array has one name and the index of the array specifies an individual element in the array. Indices start at 0 and go to the length of the array minus 1. These indices are written in square brackets following an array’s name. For example, the second place in an array called NumArr[] would be accessed using NumArr[1].

Declaration and Initialization
When an array is declared, it is given its name and data type. When it is initialized, which can be done as declared or post-phones, a size must be given to the array. The length of an array comes from either the list of values entered or the size given to an array during initialization. Each element of the array can be given its value initially or will be given the default value of that data type.

Arrays and Methods
When an array is passed in the parameters of a method, it is treated like other objects and is passed as a reference. It is important to remember that the elements of the original array are not copied. Arrays can also be returned from methods. The return type for the method is the type of array. For example, if an array of ints is being returned, the method is declared with int[] as it’s return type.

Two-Dimensional Arrays
Arrays can be viewed as rows of data, and two-dimensional arrays are arrays with rows and columns, or arrays of arrays. They often used to represent tables or board games. To declare and initialize a two-dimensional array, follow the same rules as regular arrays but insetad of one square bracket to indicate the length of the array there is one to indicate the number of rows and one for the number of colors. To get the number of rows of an array, use arrayNameHere.length. To access the number of columns, use arrayNameHere[0].length.
